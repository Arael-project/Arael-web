Arael

Arael is a lightweight CLI tool written in Go that fetches any Instagram account’s followers or following lists and lets you compare them locally.

go install github.com/Arael-project/arael-cli@latest
git clone https://github.com/Arael-project/Arael-Cli.git

After installing, run arael --help to see all commands, or have a look in the usage wiki.

Arael is still in development. New features will definitely be added soon!
