Usage examples

Arael is still in development. New features will definitely be added soon!

Arael has two primary sub-commands: fetch and compare.

1. arael fetch

Prints either the followers or the following list of one account.

arael fetch -u nasa --followers

• -u / --username — target account
• --followers — print followers
• --following — print following (default)

2. arael compare

Finds overlapping followers / following between two or more accounts.

arael compare -u nasa,esa --followers

• -u / --usernames — comma-separated list
• --followers or --following — choose one

For accounts with large number of followers/following you might need to run the script multiple times, as instagram API isnt the most reliable and most likely wont give you the whole list on the first run.

Run arael [command] --help for the full list of switches.
