Setup

One-click install

Prefer building from source? Scroll down for the go install command.

Config file headers.txt

Remember, Do NOT share your instagram headers/cookies with anyone, as it could result in compromising your account.

1 • From your browser’s DevTools copy these atributes:

User-Agent
X-Ig-App-Id
Cookie

The most important bits from the cookies are sessionid and crfstoken. Ig app id is usually not changing, so there isn't a need to swap it out that often. We reccomend pasting the whole cookies part, because what works right now might not work in the future, this way, you can ensure atleast a bit that it will be working in the future still.

If you're unsure what exactly, you can take the whole headers part. Note that when you do, you need to change the Accept-Encoding to be only gzip.

2 • Paste your headers inside a file located here: ~/.config/arael/headers.txt (below is a quick command to create the directory and the file)

mkdir -p ~/.config/arael && touch ~/.config/arael/headers.txt

Example content:
User-Agent: Instagram 310.0.0.42.119 Android
X-Ig-App-Id: 936619743392459
Cookie: sessionid=YOUR_SESSION_ID; crfstoken=YOUR_CRFS_TOKEN...

Arael loads this file automatically at start-up. If you get an error, check your headers again or paste in the whole headers segment.
